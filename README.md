# 📋 Capsitech Final Module – Project Management App  

A full-stack **Project Management System** built with:  
- **Frontend**: React + TypeScript + Bootstrap  
- **Backend**: Node.js + Express + TypeScript + MongoDB  
- **Features**: Authentication, Team Management, Project Management, Task Management  

---

## 🚀 Features
- 🔑 **Authentication** – Register, Login, JWT-based session  
- 👥 **Teams** – Add, edit, delete team members with name, email, designation  
- 📂 **Projects** – Create projects, assign multiple team members  
- ✅ **Tasks** – Manage tasks with status (To-do, In-progress, Done, Cancelled)  
- 📊 **Responsive UI** – Bootstrap-powered clean interface  
- 🔄 **Pagination & Filters** for tasks and projects  

---

## ⚙️ Backend Setup

1. Navigate to backend folder:
   
   cd backend
2. Install dependencies:

   npm install
3. Create .env file:
   PORT=5000
  MONGO_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/capsitech
  JWT_SECRET=your_jwt_secret

4.Run: 
  npx ts-node-dev src/index.ts

## ⚙️ Frontend Setup

1. Navigate to frontend folder:

    cd frontend
2. Install dependencies:

     npm instal
3. Run:
   npm run dev
